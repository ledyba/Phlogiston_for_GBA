♪名前の読みかた
　「フロジストン」です．
　「フロギストン」と読まれる方も居ますが個人的にはやはり
　「フロジストン」で読んで欲しいです^^;

♪概要
　GBAエミュレータを自動操縦してRPGのレペル上げやポケモンでの
　タマゴ孵化，なつき度上げを簡単にしよう！というソフトウェア
　です．エミュレータの速度を最大にしておくと効果バツグンです．

♪注意
　こういったツールを使って楽をするのが嫌いな人も居ます．
　ポケモンなどの対戦が出来るゲームの場合は対戦相手に許可を
　取りましょう．

♪起動までの準備について
　http://jp.sun.com/
　SunMicrosystemsのサイトから各自ダウンロードしてインストー
　ルしてインストール後，OSによって起動方法が違います．
　　-Windows
　　同じフォルダにある「Phlogiston.exe」をダブルクリックして
　　起動してください．
　　-その他のOS
　　シェルで「Phlogiston.jar」のあるフォルダに移動して
　　java -jar Phlogiston.jar
　　とすれば起動できます（多分）．

♪使い方
　起動します．そうしたらセッティングメニューからキーコンフィ
　グを選んでキーの設定をエミュレータと併せてください．
　このとき，十字キーをキーボードの十字キーにしていると，うま
　く出来ないことがあります．別のものに変えてください．
　オススメはテンキーです．
　そしてメインフレームで設定をした後，「Fire」で開始します．
　エミュレータによってはエミュレータにフォーカスを移動させな
　いと操縦できない場合があります．
　
　ちなみにファイルメニューから設定を保存することも出来ます．

　その他は適当にやってください．

♪自動操縦の原理
　JavaのRobotクラスを用いてキー入力をさせているだけです．
　メモリの内容を読んで細かい操縦がしたかったのですが，Java
　では他のプロセスのメモリを読めないらしく，断念しました．

♪このヘタクソな英語は何？
　「英語で書けたらカッコイイな〜」という事で適当に辞書を引い
　た結果こうなりました．
　間違ってるのがあったら教えてくださいです・・・．

♪「Phlogiston」の名前の由来
　燃えるものは全て「燃素」を含んでおり，燃えるというのはこの
　燃素が無くなること，そして全ての物質は「燃素」と燃えカスの
　「灰」から出来ている，というシュタールの「燃素説」です．
　燃素は英語で「Phlogiston」で，今回ソフトの名前に使わせても
　らいました．

♪ランチャーについて
　Win用のランチャーは
　http://homepage1.nifty.com/mizubashow/
　の「JavaStarter」＋ちょっぴり改造版を使用させていただきました．
　改造というのは
　「実行するクラス名を書いたファイルを「.ini」->「.jmp」に」
　「アイコン」
　です．

♪その他
　Phlogistonはフリーソフトウェアです．
　（公共の福祉に反しない限り^^;）自由に使用できます．
　実行ファイルを書き換えたりしない限り，自由に転載，二次配布
　ができます．
　もしもしたいのであれば，私と一緒に開発しませんか？

　転載時には事後でかまいませんのでメールで連絡をいただけると
